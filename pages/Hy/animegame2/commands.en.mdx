# Commands

## /account
Creates or deletes an account.\
Usage:
```sh
/account (delete/create) (username/uid)

Example for deleting accounts:
/account delete 60

Example for creating accounts:
/account create rrryfoo 202
```

## /target
Use this command before using battle/give\
Usage:
```sh
/target (uid/port)

With UID:
target 393

With port:
t
target (port of your current logged in user to target, ex. 49420)
```
## /battle
Starts a battle.\
List of enemy IDs [here](./cmds/#monster-ids)\
Usage:
```sh
/battle (enemy id)

Example:
/battle 102202001 102202001

Each enemy ID provided spawns 1 monster, so if you give 2 IDs, it will spawn 2
```

## /avatar
Gives an avatar.\
Usage:
```sh
/avatar (add/remove) (avatarId)
Example:
/avatar add 1001
```
## /item
Gives an item.\
Usage:
```sh
/item (item ID) or /item giveall
```
> The amount of items given on /item giveall may be changed on `CrepeSR\src\commands\item.ts`


## /ban
Bans a user.\
Usage:
```sh
/ban (uid/port)

Example:
/ban 29 or /ban 19191
```

## /kick
Kicks a user from the server\
Usage:
```sh
/kick (uid/port)
```

## /kickall
Kicks all users connected.\
Usage:
```sh
/kickall
```

## /debug
Literally debugs lmao\
Usage:
```sh
/debug
```
## /exit
Shuts the server down.\
Usage:
```sh
/exit
```

## /help
Sends all commands.\
Usage:
```sh
/help
```

## /maintenance
Puts the server in maintenance mode.\
Usage:
```sh
/maintenance (on/off) (maintenance message)

Example:
/maintenance on deez nuts
```

## Plane IDs
```json

```
### To find maps
Use MapEntryExcelTable.json
Which is found in CrepeSR\src\data\excel
Here, you can find plane ids & floor ids which you can use to change the scenes

### To change maps
Use GetCurSceneInfoCsReq.ts\
Which is found in CrepeSR\src\server\packets\
You will need to edit the Plane ID and Floor ID to change scenes
