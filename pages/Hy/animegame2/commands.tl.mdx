# Commands

## /account
Gumagawa o nagtatanggal ng *account*\
Paggamit:
```sh
/account (delete/create) (username/uid)

Halimbawa para sa pagtanggal ng *account*:
/account delete 60

Halimbawa para sa paggagawa ng *account*:
/account create rrryfoo 202
```

## /target
Gamitin ang command na ito bago gamitin ang *battle/give* command.
Paggamit:
```sh
/target (uid)

Halimbawa:
/target 393
```
## /battle
Nagstatart ng *battle*.\
Listahan ng mga ID ng mga *enemy* ay mahahanap [dito](./cmds/#monster-ids)\
Paggamit:
```sh
/battle (ID ng enemy)

Halimbawa:
/battle 102202001 102202001
```

## /avatar
Nagbibigay ng *avatar*.\
Paggamit:
```sh
/avatar (add/remove) (avatarId)

Halimbawa:
/avatar add 1001
```
## /item
Nagbibigay ng *item*.\
Paggamit:
```sh
/item (item ID) o /item giveall
```

## /ban
Bina-ban ang user na pinili.\
Paggamit:
```sh
/ban (uid)

Halimbawa:
/ban 29
```

## /kick
Kini-kick ang piniling *user*.\
Paggamit:
```sh
/kick
```

## /kickall
Kini-kick ang lahat ng *user* na naka konektado.\
Paggamit:
```sh
/kickall
```

## /debug
Literally debugs lmao\
Paggamit:
```sh
/debug
```
## /exit
Isinasara ang server.\
Paggamit:
```sh
/exit
```

## /help
Sinisend ang lahat ng command.\
Paggamit:
```sh
/help
```

## /maintenance
Inilalagay ang server sa *maintenance mode*.\
Paggamit:
```sh
/maintenance (on/off) (maintenance message)

Halimbawa:
/maintenance on deez nuts
```

## Plane IDs
```json
"Southern Snow Plains": "20101 (you need to go here to battle)"
```
### Para maghanap ng mga *map*
Gamitin mo ang `MapEntryExcelTable.json`\
Nahahanap ito sa `CrepeSR\src\data\excel`
Dito, mahahanap mo ang mga plane ID at floor ID na magagamit mo para baguhin ang mga *scene*

### Para magpalit ng *map*
Gamiting mo ang `GetCurSceneInfoCsReq.ts`.\
Ito ay nahahanap sa CrepeSR\src\server\packets\
Kakailanganin mong i-edit ang Plane ID at Floor ID para baguhin ang mga *scene*
