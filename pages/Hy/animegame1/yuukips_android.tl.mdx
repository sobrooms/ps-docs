# Pagkokonekta sa Android GI PS
<Callout type="info">
  **Ang pamamaraang ito ay gumagamit ng isang pinalitan ang pangalan ng folder package name, kaya kailangan mo ang opisyal na data ng laro mula sa [opisyal na app](https://play.google.com/store/apps/details?id=com.miHoYo.GenshinImpact&hl=en_US&gl=US).**
</Callout>

## Mga Kinakailangan
1. I-download ang isa sa mga [modded na launcher](https://docs.ps.rrryfoo.cf/GenshinImpact/wyuukips_android#modded-launchers-apk)
2. Download at i-install ang opisyal na laro. **Siguraduhin na i-download mo muna ang data ng laro sa pamamagitan ng paglalaro sa opisyal na server bago maglaro sa private server (o lamang atleast pumunta lamang sa laro at i-tap kapag naglo-load ang laro, upang i-download ang mga file ng laro).**. You may also download audio packs if you would like. (in-game's *Audio* tab in settings).
3. Download at i-install ang "Ziparchive".
4. I-buksan mo ang *Ziparchive*  at hanapin ang folder ng data ng laro sa **`storage/emulated/0/Android/data`** at makahanap ng isang folder na may pangalang **`com.miHoYo.GenshinImpact`**.
5. Palitan mo ang pangngalan ng folder (`com.miHoYo.GenshinImpact` to `com.miHoYo.GenshinImpact.backup`). (*Kailangan ng root access ang mga user sa Android 13, hindi kailangan ng root access ang mga user na nasa Android 11-12*)
6. Pagkatapos nito, i-uninstall ang opisyal na application (hindi ang folder ng laro, ngunit ang aktwal na application na pinangalanang "Genshin Impact" sa iyong home screen). Pagkatapos mo i-delete ang folder, i-install ang isa sa mga [modded app launchers](https://ps-docs.theonesaya.repl.co/GenshinImpact/wyuukips_android#modded-launchers-apk).
7. Buksan muli ang Ziparchive at hanapin ang backup ng folder ng data ng laro, na matatagpuan pa rin sa **`/storage/emulated/0/Android/data`** (the one that you should have named `com.miHoYo.GenshinImpact.backup`)
8. Palitan ang pangalan ng backup folder pabalik sa orihinal na pangalan. (ang orihinal na pangalan ng folder ay: `com.miHoYo.GenshinImpact`)
9. I-buksan ang *Genshin Impact*.

## Mga madalas itinatanong
<Callout type="info" emoji="ðŸ’¬">
  **Q: "Bakit ako natitigil sa 'Preparing to load data - 0.00%'?"**\
  *A: : Kung talagang nabasa mo ang lahat, kailangan mo munang i-download ang data ng laro mula sa opisyal na laro. Hindi mo maaaring laktawan ito. Bilang kahalili, kung hindi mo nais na mag download mula sa opisyal na laro, maaari kang mag download ng data ng laro mula sa ibang lugar, o mula sa isang backup ng iyong iba pang cellphone...*
</Callout>
<Callout type="info" emoji="ðŸ’¬">
  **Q: "Ano ang pagkakaiba ng V1 sa V2?"**\
  *A:  Ang V1 ay maaari lamang kumonekta sa YuukiPS server.
  Maaari din kumonekta ang V2 sa localhost, opisyal na server (minsan), at YuukiPS. Gayundin 0kb download speed ay naayos sa update na ito. (Puwede i-laro ang opisyal na server sa V2 lamang para sa mga rooted na *device* (kung mayroon silang kakayahan (*skill*) upang malaman kung paano gawin ito)
</Callout>
<Callout type="info" emoji="ðŸ’¬">
  **Q: ""Ano po ang pagkakaiba ng GIO sa Grasscutter Server?"**\
  *A: Ang GIO ay isang clone ng opisyal na server, na nangangahulugang ang lahat ng mga function ay gumagana nang walang anumang mga bug at quests. Lahat ng gumagana ngunit maaari lamang tumakbo sa 3.2.
  Ang server ng Grasscutter ay isang emulator ng server na maaaring tumakbo sa mga pinakabagong bersyon at beta na bersyon, ngunit maraming mga function ang hindi gumagana, ito ay lamang na ang mga command ay mas madaling i-run dahil maaari silang ma access sa pamamagitan ng bot chat ni Ayaka.
</Callout>
<Callout type="info" emoji="ðŸ’¬">
  **Q: "Paano ko aayusin ang '[Failed to load il2cpp](https://github.com/YuukiPS/Launcher-Android/issues/3)'?"**\
  *A:  Ayusin ito sa pamamagitan ng pagtanggal ng folder na may pangalang `Unity` at `il2cpp` sa folder ng data ng laro ng Genshin.*
</Callout>
<Callout type="info" emoji="ðŸ’¬">
  **Q: "Paano ko aayusin ang 'Error 4214'?"**\
  *A:  Kung pumasok ka sa opisyal na server: Ang apk launcher mod ay hindi maaaring kumonekta sa opisyal na server, dahil ito ay mapanganib.
  Kung pumasok ka sa pribadong server: Ang server ay down o ang iyong *proxy* ay hindi gumagana.*
</Callout>
